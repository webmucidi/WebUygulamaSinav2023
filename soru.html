WTUG DERSİ 1.DÖNEM UYGULAMA SINAVI

Uygulama Yönergesi:

  1. Aşağıda ekran görüntüsü verilen Layout’u CSS Grid yöntemiyle oluşturunuz.(25 puan)
  1. Ana iskelet Grid sözel yöntemle yapılacaktır.
  2. Orta alandaki kutular Grid sayısal yöntemle yerleştirilecektir.
  3. Ana iskelet için Semantic etiketleri kullanılacaktır. Kutular için istediğiniz nesne yapısını kullanabilirsiniz.
  4. Layout için mobil uyumlu yapı oluşturulacaktır.

  1. Aşağıdaki ekran görüntüsünde belirtilen şekilde içerik ekleme ve dizilişlerini CSS Flexbox yöntemiyle yapınız. (25 puan)
  1. Header alanında kullanılacak resim “logo.png” uygulama klasöründe mevcuttur.
  2. Kutulardaki resimlerin ilki “css.jpeg” içerik resmi olarak kutuya eklenecek ve kutuya sığması sağlanacaktır.
  3. Diğer resim “background.png” aşağıdaki kutuya arkaplan resmi olarak eklenecek ve arkaplanı kaplaması sağlanacaktır.
  4. Tüm kutulardaki içeriklerin yatay ve dikey hizalamaları ekran görüntüsündeki şekilde olacaktır.
  5. Kutularda uygulanması istenen ileri CSS uygulamaları(opacity,transition vs)için gerekli komutlar ekran görüntüsünde belirtilmiştir. İlgili kutuya uygulanacaktır.
